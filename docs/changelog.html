Selah Focus

Version History

All notable changes to Selah Focus are documented here.

1.0.1

February 2026

This release adds custom verse management and fixes timer persistence when the popup is closed during an active session.

  • ✨ Custom Verses – Users can now add, edit, and delete their own Bible verses. Custom verses appear alongside curated verses in rotation, with full theme and translation support.
  • ✨ Persistent Timer – The timer now seamlessly reconnects when the popup reopens after being closed. If a focus or break session is active, the timer continues from the correct elapsed time.
  • 📝 Custom Verse Form – New form component for adding verses with text, reference, theme, and translation. Validation ensures data integrity.
  • 📋 Custom Verse List – Displays all saved custom verses with edit and delete actions. Integrated into the Settings tab for easy management.
  • 🔧 Storage Resilience – Improved Chrome storage fallback to localStorage for testing environments while maintaining sync across Chrome devices.
  • ✅ Test Coverage – Comprehensive tests added for custom verse management, integration scenarios, and persistent timer behavior.

1.0.0

February 2026

Initial public release. Selah Focus combines the Pomodoro Technique with Scripture-based focus rhythms for distraction-free work anchored in the Word.

  • ⏱️ Pomodoro Timer – Customizable focus sessions (default 25 min), short breaks (5 min), and long breaks (15 min) with configurable cycle tracking.
  • 📖 Scripture Integration – 55 curated Bible verses organized by theme (Peace, Gratitude, Strength, Purpose, Trust, Growth) with support for ESV, NIV, and KJV translations.
  • 🎯 Pre-Focus Verse Display – A 10-second Scripture preview before each focus session to ground users in the Word.
  • 💭 Reflection Prompts – Thoughtful prompts during breaks encourage deeper meditation on Scripture.
  • ⚙️ Full Customization – Adjust all timer durations, enable/disable Scripture, choose translation and theme. Settings persist across Chrome devices.
  • 🔔 Smart Notifications – Desktop notifications when sessions complete. Extension badge shows active focus status.
  • ⌨️ Keyboard Navigation – Space (pause/resume), Enter (start), R (reset), S (skip break), Alt+S (settings), Escape (close).
  • ♿ Accessibility – Full WCAG 2.1 AA compliance with keyboard focus indicators and screen reader announcements at meaningful milestones.
  • 🔐 Privacy First – Zero tracking, no analytics, no external APIs. All data stored locally on the device.
